# **VeilaLight™ \- Zero-G Projector Backrest Demo**

这是一个基于 WebGL (Three.js) 的 3D 产品交互展示页面。演示了 VeilaLight 零重力投影靠背的概念设计，包含交互式的材质切换和爆炸视图动画。

## **✨ 功能特性**

* **3D 交互**：支持 360° 旋转、缩放和平移查看产品细节。  
* **三种视图模式**：  
  * **透视 (X-ray)**：半透明外壳，展示内部骨架结构。  
  * **拟真 (Solid)**：模拟真实的深炭灰磨砂与深空灰金属质感。  
  * **爆炸视图 (Exploded)**：一键拆解所有零部件，展示内部风道、电路板与光机结构。  
* **动态交互**：  
  * **角度调节**：通过拖动左侧的角度盘调节靠背角度（90° \- 180°）。  
  * **智能高亮**：鼠标悬停在爆炸视图的零件上时，会高亮显示并弹出详细组件说明。  
* **光影效果**：集成了环境光、方向光和半球光，以及呼吸灯特效。

## **🛠️ 技术栈**

* **HTML5 / CSS3**：单文件结构，无需构建工具。  
* **Three.js (r128)**：核心 3D 渲染引擎。  
* **Tween.js**：处理爆炸视图和平滑复原的补间动画。  
* **OrbitControls**：处理摄像机交互。
